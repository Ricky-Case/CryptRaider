# CryptRaider
Developed as part of the GameDeev.tv "Unreal Engine 5 C++ Developer" Udemy course.

The primary focus of this project was to learn how to develop the following using Unreal Engine 5:
- Debugging
- Geometry
- Lighting (Dynamic)
- Map Design
- Physics Puzzles
- Tags

# Required Package(s)
- Medieval Dungeon
